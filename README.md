# Proyecto SMR - Sistema de Usuarios

## Descripción
Este proyecto consiste en la implantación de un sistema de gestión de usuarios en una empresa pequeña utilizando Windows Server. Se configura un dominio local donde se administran usuarios, grupos y permisos de acceso a recursos compartidos.

Además, se desarrolla una pequeña interfaz web en HTML y CSS que simula un panel de administración para visualizar la estructura del sistema.

## Objetivos
- Crear un dominio local
- Gestionar usuarios y grupos
- Aplicar permisos y medidas de seguridad
- Representar el sistema mediante una interfaz web

## Tecnologías
- Windows Server
- HTML y CSS
- VirtualBox
