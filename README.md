# Proyecto SMR - Sistema de Usuarios en Empresa

## 📌 Descripción
Este proyecto consiste en la implantación de un sistema de gestión de usuarios en una empresa simulada utilizando Windows Server.

Se ha creado un entorno con un dominio local donde se gestionan usuarios, grupos y permisos de acceso a recursos compartidos, simulando una infraestructura real de empresa.

Además, se ha desarrollado una interfaz web en HTML, CSS y JavaScript que representa visualmente el sistema de usuarios y su organización.

---

## 🎯 Objetivos
- Crear un dominio local funcional
- Gestionar usuarios y grupos mediante Active Directory
- Aplicar permisos de acceso a recursos compartidos
- Mejorar la seguridad mediante políticas de grupo (GPO)
- Representar el sistema mediante una interfaz web
- Simular un entorno empresarial real

---

## 🖥️ Estructura del sistema
El sistema está compuesto por:

- Servidor Windows Server (Controlador de dominio)
- Cliente Windows 10 unido al dominio
- Dominio: empresa.local
- Usuarios organizados por departamentos (IT, RRHH, Dirección)

---

## 🛠️ Tecnologías utilizadas
- Windows Server
- Active Directory
- GPO (Group Policy)
- HTML, CSS y JavaScript
- VirtualBox
- Figma (diseño de interfaz)

---

## 🌐 Desarrollo web
Se ha desarrollado una interfaz web como complemento del sistema.

- Primera versión básica en HTML y CSS
- Posteriormente rediseñada para mejorar la estética
- Diseño previo realizado en Figma
- Mejora visual y estructural para un resultado más profesional

---

## 🔐 Seguridad
Se han aplicado medidas de seguridad como:

- Gestión de permisos mediante grupos
- Restricción de acceso a carpetas compartidas
- Aplicación de políticas de grupo (GPO)
- Control de contraseñas y acceso
